## Трудности и их решение

| Трудность | Решение |
|-----------|---------|
| Сетевые ограничения при локальном запуске Telegram-бота (таймауты, блокировка api.telegram.org) | Использован VPN. Код бота адаптирован с увеличенными тайм-аутами (`connect_timeout=30`). |
| Низкая вовлечённость части команды, перенос задач | Перераспределение задач: невыполненные работы взяты на себя или переданы другим участникам. |
| Сложность с настройкой прокси для бота | Отказ от прокси в пользу VPN, что обеспечило стабильное соединение. |
