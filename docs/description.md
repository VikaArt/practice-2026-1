# Описание проекта «Карты компетенций для выпускника»

## Название проекта
Карты компетенций для выпускника

## Цель проекта
Повысить лояльность студентов к прохождению тестов РСВ и процент прохождения курсов от РСВ, предоставив понятный цифровой инструмент для оценки и развития компетенций.

## Задачи проекта
- Провести опрос студентов по осведомлённости о тестировании РСВ
- Реализовать контент-стратегию и привлечь аудиторию в Telegram-канал
- Обеспечить запуск сайта и Telegram-бота
- Создать ВК-сообщество и ВК-бота
- Проработать курсы для развития компетенций
- Оформить проектную документацию

## Продукты проекта
- **Telegram-бот** (тест от РСВ, описание компетенций, игра в правда/миф, оставление фидбека)
- **Telegram-канал** (5 рубрик, регулярные посты)
- **ВК-сообщество** (дублирование контента)
- **ВК-бот**

## Заказчик
Центр компетенций Московского политехнического университета

## Партнёры
Россия – страна возможностей (РСВ), hh.ru
